# Silent Coworking Space Website

A premium, highly interactive, and responsive landing page for **Silent Coworking Space** located in Mota Varachha, Surat. 

This website is custom-built using HTML5, Vanilla CSS3, and JavaScript, featuring modern typography, glassmorphism aesthetics, scroll reveal animations, and direct contact form submission integration to WhatsApp.

## 🚀 Key Features

* **Calm & Premium Aesthetics**: Tailored color scheme featuring Emerald Green (`#059669`) for focus and Warm Charcoal Slate (`#0F172A`) for text.
* **Interactive Plan Selector**: Clicking **"Select Plan"** on any membership card automatically fills the contact form query field with the corresponding package details and smoothly scrolls the user to the contact form.
* **Instant WhatsApp Integration**: Submitting the contact form performs validation checks and opens a pre-formatted, bold message directly in WhatsApp to `+91 7863807008` containing the user's name, phone number, email, and query details.
* **Fully Responsive**: Adapts seamlessly to all screen sizes (mobile devices, tablets, laptops, and ultra-wide desktops) with an elegant mobile drawer navigation menu.
* **Vite Powered**: Running a hot-reloading dev environment for efficient development.

## 🛠️ Tech Stack

* **Core**: HTML5, Vanilla JavaScript (ES6)
* **Styling**: Vanilla CSS3 (Custom variables, Flexbox/CSS Grid, and transitions)
* **Tooling**: Vite (Development server)

## 💻 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Run
1. Clone or download this repository.
2. Open your terminal in the directory and run:
   ```bash
   npm install
   ```
3. Start the local development server:
   ```bash
   npm run dev
   ```
4. Build the production bundle:
   ```bash
   npm run build
   ```
